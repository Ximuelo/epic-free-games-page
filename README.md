## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ximuelo/epic-free-games-page.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the project as a development server
   ```js
   npm run dev
   ```
Open http://localhost:3000 with your browser to see the result.
